Desenvolvimento de Portfólio Pessoal Online

Projeto Completo de Banco de Dados: Modelagem e Implementação

Descrição: Desenvolvimento abrangente de um projeto de Banco de Dados, englobando desde a modelagem conceitual até a implementação. Incluiu a elaboração de um Modelo Entidade-Relacionamento (MER) para uma Rede de Hotéis, identificando entidades, atributos e relacionamentos. Posteriormente, foi realizada a implementação em MySQL Workbench para uma Locadora de Veículos, com criação de tabelas (DDL), população de dados e desenvolvimento de consultas SQL (DML) para gerenciamento de operações e análises de dados.

Tecnologias Utilizadas: MySQL, SQL (DDL, DML), Modelagem de Dados (MER), Conceitos de Banco de Dados Relacionais.

Trabalho de Lógica de Programação e Algoritmos (LPA)

Descrição: Desenvolvimento de um trabalho prático sobre Lógica de Programação e Algoritmos. O projeto envolveu a criação de algoritmos em Python para resolver problemas de cálculo de desconto em vendas, interface para pedidos de açaí, sistema de cobrança de copiadora e gerenciamento de livros. Foram aplicados conceitos de estruturas condicionais (if/elif/else), laços de repetição (while/break/continue), funções, tratamento de erros (try/except) e estruturas de dados (listas e dicionários).

Tecnologias Utilizadas: Python, Lógica de Programação, Algoritmos, Estruturas de Dados.
